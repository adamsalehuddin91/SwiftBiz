# SwiftApps — Freelance Pricing Guide
**Updated:** 2026-04-04 | **Market:** Malaysian SME (Melaka focus)

> **Penting:** Ada DUA model berbeza — pilih SATU untuk setiap client.
> - **One-Off** = Bayar sekali, sistem milik client selamanya
> - **SaaS Monthly** = Setup fee (sekali) + yuran bulanan (ongoing)

---

## MODEL PRICING

### Model 1: One-Off (Beli Putus)
Client bayar lump sum → sistem terus milik diorang. Sesuai untuk client ada modal.

### Model 2: SaaS Monthly (Sewa Sistem)
Client bayar setup fee sekali + monthly fee. Sesuai untuk client modal kecil.
- **Setup fee** = cover masa deploy, configure, train client (bukan bayar sistem)
- **Monthly fee** = hosting + support + minor updates

## SYARAT BAYARAN

**Format standard (semua project):**

| Milestone | % | Bila |
|-----------|---|------|
| Deposit | 50% | Sebelum start kerja |
| Progress | 30% | Lepas demo / draf sistem diluluskan client |
| Handover | 20% | Sebelum serah password / live launch |

> **Tiada bayaran penuh sebelum kerja siap.** Deposit melindungi masa kau, baki melindungi client.

---

## CARA APPROACH CLIENT (SaaS Conversion Strategy)

```
1. Tunjuk demo sistem (15 minit) — biar diorang nampak real product
2. Offer One-Off dulu → tutup deal, collect deposit
3. Build & launch sistem
4. 3 bulan lepas launch → WhatsApp semula
   "Saya ada maintenance package, RM X/bulan,
    termasuk hosting + support + update kecil"
5. Diorang dah guna & trust → senang convert ke monthly
```
