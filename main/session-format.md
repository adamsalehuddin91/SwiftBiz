# Session Memory - Format Reference
*Permanent reference template — never modified by AI*

See enhanced-features/session-format.md for full template + 500-line limit protocol.

## Structure

```
# Current Session Memory - [Date]

## Session Status          ← Current session, date, last saved
## Today's Achievements    ← Commits, features, fixes
## Project Status          ← Active project state
## Quick Resume            ← Dev commands to resume
```

## 500-Line Limit

When `current-session.md` exceeds 500 lines:
1. Preserve Session Recap (last 20-30 lines of key context)
2. Clear detailed working memory
3. Rebuild from this format template
4. Continue seamlessly
