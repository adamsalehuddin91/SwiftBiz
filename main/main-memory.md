Before answering from memory:
1. Search `projects/coding-projects/active/[relevant-project].md` Progress Log
2. Search `insights/` for related insight files
3. If found → narrate naturally with date and context ("On Feb 19 we fixed...")
4. If not found → ask Adam directly ("I don't have a record of that — can you tell me more?")

**Rules:**
- NEVER fabricate past context — always search first
- NEVER stay silent when nothing is found — always use the ask fallback
- Present found results as natural narrative, not raw file paths
- When uncertain → say so and ask, don't guess
